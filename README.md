# ROADWATCH
Smart Roads, Smarter Governance
ROADWATCH is an AI-powered civic platform that enables users to report road damage such as potholes using real-time image capture with automatic GPS detection. The system analyzes road conditions using computer vision and generates a road health score along with predictive insights for future damage.

It enhances transparency by linking each road to its contractor and budget details, allowing users to track accountability. The platform also introduces a contractor rating system based on road quality and a municipality performance tracker based on repair time.

With an interactive map interface and an admin dashboard, ROADWATCH creates a complete ecosystem for smart infrastructure monitoring and citizen engagement.
